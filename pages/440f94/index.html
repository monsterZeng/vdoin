# 拉普拉斯矩阵

假设G是一个图。定义图上的拉普拉斯矩阵为L(G)=Δ(G)−A(G)L(G)=\Delta(G)-A(G)L(G)=Δ(G)−A(G)。这里A(G)A(G)A(G)​为邻接矩阵，Δ(G)\Delta(G)Δ(G)为度矩阵。

拉普拉斯除了有领接矩阵相同的属性

  • 值为-1对应的列节点为入度节点。
  • 对角线上的值表示入度节点的个数。

还有其他属性。
