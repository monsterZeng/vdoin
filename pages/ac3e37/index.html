梯度消失和爆炸

#

# 原因

层数比较多的神经网络模型在使用梯度下降法对误差进行反向传播时会出现梯度消失和梯度爆炸问题。梯度消失问题和梯度爆炸问题一般会随着网络层数的增加变得越来越明显。

由于梯度是反向传播回来的，如果在传播过程中梯度消失了，那么最前面几层的参数就没有被训练到，仅仅是做了映射。此时只有后面几层参与了学习。

如果传播过程中，梯度爆炸了，那么前面几层的参数更新非常剧烈，有可能溢出。

# 判断

  1. 梯度消失情况下，模型的参数无法从训练数据中得到更新。
  2. 梯度爆炸情况下，模型的参数剧烈变化。
  3. 训练过程中，损失不会变化；损失剧烈上下剧烈抖动。

# 推导

# 1、从深层的网络角度看梯度消失和梯度爆炸

假设有三层隐藏层的神经网络，激活函数使用sigmoid。fi(x)f_i(x)f​i​​(x)是带激活函数的输出。

# 激活函数角度

那么为什么会出现梯度消失的现象呢？因为通常神经网络所用的激活函数是sigmoid函数，这个函数有个特点，就是能将负无穷到正无穷的数映射到0和1之间，并且对这个函数求导的结果是f′(x)=f(x)(1−f(x))。因此两个0到1之间的数相乘，得到的结果就会变得很小了。神经网络的反向传播是逐层对函数偏导相乘，因此当神经网络层数非常深的时候，最后一层产生的偏差就因为乘了很多的小于1的数而越来越小，最终就会变为0，从而导致层数比较浅的权重没有更新，这就是梯度消失。

假设使用了sigmoid函数

σ(x)=11+exp(−x)\sigma(x) = \frac{1}{1+\exp{(-x)}} σ(x)=​1+exp(−x)​​1​​

sigmoid的梯度为

σ′(x)=σ(x)(1−σ(x))=−(σ(x)−12)2+14\begin{aligned} \sigma'(x)&=\sigma(x)(1-\sigma(x)) \\ &=-(\sigma(x)-\frac{1}{2})^2+\frac{1}{4} \end{aligned} ​σ​′​​(x)​​​​=σ(x)(1−σ(x))​=−(σ(x)−​2​​1​​)​2​​+​4​​1​​​​

可见sigmoid的梯度最大值为1/4，通常初始化的网络权重都小于1，也就是∣σ(z)′w∣≤14|\sigma(z)'w|\le\frac{1}{4}∣σ(z)​′​​w∣≤​4​​1​​。所以很容易引起梯度消失。但是，如果初始化的网络权重很大，有可能导致∣σ(z)′w∣>1|\sigma(z)'w|>1∣σ(z)​′​​w∣>1，那么就会产生梯度爆炸。

# 解决方法

  • 使用ReLU函数，正数部分的梯度恒等于1。
    • 优点：1、解决了梯度消失和爆炸；2、计算简单，速度快，加速网络训练。
    • 缺点：1、负数恒等于0，导致某些神经元无法激活；2、输出不是以0为中心。
  • 其它ReLU函数，解决了ReLU的缺点，但是速度没有ReLU快。
  • 长短期记忆网络，使用门控网络。
  • 梯度截断，是针对梯度爆炸提出来的。
  • 权重正则，惩罚较大权重值的损失函数。
  • BN和LN
  • 残差结构，针对梯度消失被提出来的。
