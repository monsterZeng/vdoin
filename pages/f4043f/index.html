# Accuracy

准确率是最简单的的评价标准。具体为：在所有样本中，预测正确的的个数

Accuracy=TN+TPTN+FN+TP+FP\text{Accuracy}=\frac{TN+TP}{TN+FN+TP+FP} Accuracy=​TN+FN+TP+FP​​TN+TP​​

准确率非常简单，但是在正负样本不均衡时，这个结果可能存在虚假情况。例如，当正样本个数为99，负样本个数为1。那么不需要去建立模型什么的，只需要将遇到的样本全部分类为正，那么准确率都可以达到99%。所以，准确率的考差不够全面。

# Precision

精确率是针对预测样本而言的。预测为正样本的个数中，实际为正样本的概率。

Precision=TPTP+FP\text{Precision}=\frac{TP}{TP+FP} Precision=​TP+FP​​TP​​

# Recall，Sensitivity

召回率是针对原样本而言的。预测为正样本的个数占总的正样本的比率

Recall=TPTP+FN\text{Recall}=\frac{TP}{TP+FN} Recall=​TP+FN​​TP​​

# F1

F1值是精确率和召回率的调和平均值

F1=2∗Precision*RecallPrecision+Recall\text{F1}=2*\frac{\text{Precision * Recall}}{ \text{Precision + Recall}} F1=2∗​Precision+Recall​​Precision*Recall​​

# ROC & AUC

AUC是ROC曲线与X轴形成的面积。

ROC曲线可以用来在阈值变动时，检测分类器的性能。

ROC曲线以 true positive rate(真阳性，TPR)和 false positive rate(假阳性，FPR)为Y、X轴。对预测样本的概率进行从高到底排序，然后用一个变化的阈值去形成点(TPR,FPR)，最后将这些点连成线，就形成了ROC曲线。

TPR=TPP,FPR=FNN\text{TPR}=\frac{TP}{P}, \text{FPR}=\frac{FN}{N} TPR=​P​​TP​​,FPR=​N​​FN​​

ROC曲线如图所示。

Link (opens new window)

判断ROC曲线的好坏，是ROC的曲线与X轴形成的面积。面积越大，那么，分类器性能越好。什么时候ROC的曲线的面积越大呢？如上图所示，当ROC曲线越靠近左上角时，ROC曲线面积越大。那么什么时候ROC曲线越往左上方靠近呢？就是在一个阈值里，TPR越大，FPR越小。所以我们希望分类器将预测为正的样本概率越大越好，预测为负的样本越小越好，将两个样本分得足够开。那么在同一个阈值下，TPR就会越高，FPR就会越低。AUC就越大。
