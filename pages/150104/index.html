Default arguments

# Default arguments

A default argument is a default value provided for a function parameter.

​ 如果用户未给带有默认参数的参数提供显式参数，则将使用默认值。如果用户为参数提供了实参，则使用用户提供的值。

​ 由于用户可以选择提供特定的参数值还是使用默认值，因此通常将提供默认值的参数称为可选参数。

void printValues(int x, int y=10) // 10 is the default argument, y is now an optional parameter
{
    std::cout << "x: " << x << '\n';
    std::cout << "y: " << y << '\n';
}
 
int main()
{
    printValues(1); // y will use default argument 10          --output: 1, 10
    printValues(3, 4); // y will use user-supplied argument 4  --output: 3, 4
}

Multiple default arguments

​ 一个函数可以具有多个默认参数：

void printValues(int x=10, int y=20, int z=30)
{
    std::cout << "Values: " << x << " " << y << " " << z << '\n';
}
printValues(1, 2, 3);
printValues(1, 2);
printValues(1);
printValues();

​ 请注意，如果没有同时为参数x和y提供默认值，则不可能为参数z提供默认值。这是因为C ++不支持函数调用语法，例如printValues(,, 3)。这有两个主要结果：

​ 1)所有默认参数都必须是最右边的参数。不允许以下内容：

void printValue(int x=10, int y); // not allowed
1

​ 2）如果存在多个默认参数，则最左边的默认参数应该是用户最有可能明确设置的参数。

Default arguments can only be declared once

​ 声明后，将无法重新声明默认参数。这意味着对于具有前向声明和函数定义的函数，可以在前向声明或函数定义中声明默认参数，但不能两者都声明。

void printValues(int x, int y=10);
 
void printValues(int x, int y=10) // error: redefinition of default argument
{
    std::cout << "x: " << x << '\n';
    std::cout << "y: " << y << '\n';
}

Default arguments and function overloading

​ 具有默认参数的函数可能会重载。例如，允许以下内容：

void print(std::string string);
void print(char ch=' ');
1
2

如果用户要调用print()，它将解析为print(' ')，这将打印一个空格。

​ 但是，重要的是要注意，可选参数不计入使函数唯一的参数。因此，以下内容是不允许的：

void printValues(int x);
void printValues(int x, int y=20);
1
2

​ 如果调用者要调用printValues(10)，则编译器将无法消除用户是否希望使用默认值printValues(int)或printValues(int，20)。

Summary

​ 默认参数提供了一种有用的机制来指定用户可以选择为其提供值的参数。

Rule: If the function has a forward declaration, put the default argument there. Otherwise, put them in the function definition.
