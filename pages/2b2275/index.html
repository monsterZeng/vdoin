只出现一次的数

尽管只出现一次数有多种变形题，都是数组中只有一个出现过一次的数，其他数都是出现个k次。

  • # 只出现一次的数字 (opens new window)

  • # 只出现一次的数字 II (opens new window)

如果是偶数，那么可以使用异或运算。

class Solution {
public:
    int singleNumber(vector<int>& nums) {
        int ans = 0;
        for(int i = 0; i < nums.size(); i ++){
            ans ^= nums[i];
        }
        return ans;
    }
};

如果是奇数，或者未知。那么可以将每个数的每一位拆分。一个数出现k次，那么同一个为必将出现k次。

class Solution {
public:
    /**
     * 代码中的类名、方法名、参数名已经指定，请勿修改，直接返回方法规定的值即可
     *
     * 
     * @param arr intvector 
     * @param k int 
     * @return int
     */
    int foundOnceNumber(vector<int>& arr, int k) {
        vector<int>nums(32);
        for(int i = 0; i < arr.size(); i ++){
            for(int j = 0; j < 32; j ++){
                nums[j] += ((arr[i] >> j) & 1);
            }
        }
        int sum = 0;
        for(int i = 0; i < 32; i ++){
            nums[i] %= k;
            sum += (nums[i] << i);
        }
        return sum;
    }
};

# 只出现一次的数字 III (opens new window)

给定一个整数数组 nums，其中恰好有两个元素只出现一次，其余所有元素均出现两次。 找出只出现一次的那两个元素。你可以按 任意顺序 返回答案。

这里有两个数字都出现了一次。一次异或不能够区分出两个数字来。但是如果我们能把里面的数分成两部分，两个部分仅存在一个出现一次的数字，其余数字都出现两次。

我们假设出现一次的两个数是a，b。我们可以利用对所有数字进行异或的结果，因为我们知道。对所有数字的异或，仅仅是a^b的结果。这个结果中的位为1，表示a和b在这一位的不同。所以，我们可以取某一位，然后利用这一位将这个数组分成两部分。然后分别异或。

注意，这个结果不可能为0，因为如果为0，那么a==b。但是先前条件是a != b。

class Solution {
public:
    vector<int> singleNumber(vector<int>& nums) {
        int ret = 0;
        // 获得异或结果
        for(int i = 0; i < nums.size(); i++){
            ret ^= nums[i];
        }
        // 找到这个异或结果中的不为零的一位
        int i = 0;
        for(; i < 32; i ++){
            if(((ret >> i) & 1) == 1){
                break;
            }
        }
        // 分组异或
        int ret1 = 0, ret2 = 0;
        for(int j = 0; j < nums.size(); j ++){
            if(((nums[j] >> i) & 1) != 0){
                ret1 ^= nums[j];
            }else{
                ret2 ^= nums[j];
            }
        }
        return vector<int>{ret1, ret2};
    }
};
