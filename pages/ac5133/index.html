Layer Normalization

# Layer Normalization

# Abstract

However, the effect of batch normalization is dependent on the mini-batch size and it is not obvious how to apply it to recurrent neural networks. In this paper, we transpose batch normalization into layer normalization by computing the mean and variance used for normalization from all of the summed inputs to the neurons in a layer on a single training case.

# Layer normalization

注意，一层输出的变化往往会导致下一层求和输入的高度相关变化，特别是ReLU单元的输出可能会发生很大变化。这表明“协变量移位”问题可以通过固定每一层求和输入的平均值和方差来减少。因此，我们对同一层中所有隐藏单元的层归一化统计量计算如下:

μl=1H∑i=1Hailσl=1H∑i=1H(ail−μl)\begin{aligned} \mu^l&=\frac{1}{H}\sum_{i=1}^{H}a_i^l \\ \sigma^l &= \sqrt{\frac{1}{H}\sum_{i=1}^{H}(a_i^l-\mu^l)} \end{aligned} ​μ​l​​​σ​l​​​​​=​H​​1​​​i=1​∑​H​​a​i​l​​​=​⎷​​​​​​​H​​1​​​i=1​∑​H​​(a​i​l​​−μ​l​​)​​​​​

H指的是一层隐向量的维度。注意，layer normalization是在每个训练数据上做normalization。

batch normalization用在RNN的一个很重要的问题就是测试期间的 μ\muμ 和 σ\sigmaσ 是从训练期间获得的，也就是说每个维度的进行正则是用训练期间的参数。但是有可能期间的序列长度都小于训练期间的，那么导致统计信息失效。并且，强行将每个维度的数据进行normalization，很可能破坏序列之间的差异性。

在一个标准的RNN中，在每个时间步长中，对循环单元的求和输入的平均幅度有增长或收缩的趋势，导致梯度爆炸或消失。在一层归一化RNN中，归一化项使其在重新缩放一层的所有求和输入时保持不变，这导致了更稳定的隐藏到隐藏的动态。
