# NLLoss

负对数似然损失。用C类训练分类问题是有用的。

# BCELoss

l(x,y)=L={l1,...,lN}T,li=−wn[yn∗logxn+(1−yn)∗log(1−xn)]\mathcal{l}(x,y)=L=\{l_1,...,l_N\}^T,l_i=-w_n[y_n*\log {x_n} + (1-y_n)*\log{(1-x_n)}] l(x,y)=L={l​1​​,...,l​N​​}​T​​,l​i​​=−w​n​​[y​n​​∗logx​n​​+(1−y​n​​)∗log(1−x​n​​)]

这里x是预测值，y是真实标签。当x是0或者为1时，有可能会导致数值上的不稳定，因为\underset{x \rightarrow 0}{\lim}\log{(x)}=-\infty​。所以有时候需要将对数函数大小规定在大于某个数里。Pytorch将log函数的输出规定大于等-100。

\underset{x \rightarrow 0}{\lim}\log{(x)}=-\infty

# BCEWithLogitsLoss

这种损失将Sigmoid层和BCELoss合并在一个类中。这个版本在数值上比使用普通的Sigmoid和BCELoss更稳定，因为通过将操作合并到一个层中，利用了log-sum-exp技巧来获得数值稳定性。

l(x,y)=L={l1,...,lN}T,ln=−wn[yn∗logσ(xn)+(1−yn)∗log(1−σ(xn))]l(x,y)=L=\{l_1,...,l_N\}^T,l_n=-w_n[y_n*\log{\sigma(x_n)}+(1-y_n)*\log{(1-\sigma(x_n))}] l(x,y)=L={l​1​​,...,l​N​​}​T​​,l​n​​=−w​n​​[y​n​​∗logσ(x​n​​)+(1−y​n​​)∗log(1−σ(x​n​​))]

# CrossEntropyLoss

结合了LogSoftmax和NLLoss

loss(x,class)=−logexp(x[class])∑jexpx[j]=−x[class]+log∑jexp(x[j])\text{loss}(x,class)=-\log{\frac{\exp(x[class])}{\sum_j\exp{x[j]}}}=-x[class]+\log{\sum_j\exp(x[j])} loss(x,class)=−log​∑​j​​expx[j]​​exp(x[class])​​=−x[class]+log​j​∑​​exp(x[j])

从公式中可以看出，如果当前想要当前类别判断成功，loss尽可能小，那么就真实类别的概率要足够大，而其他类别的概率要足够小。能够提高预测类别的置信度。

# Focal Loss

L={−α(1−y′)γlogy′,y=1−(1−α)y′γlog(1−y′),y=0L= \begin{cases} -\alpha(1-y')^{\gamma}\log{y'}, y=1 \\ -(1-\alpha){y'}^{\gamma}\log{(1-y')}, y = 0 \end{cases} L={​−α(1−y​′​​)​γ​​logy​′​​,y=1​−(1−α)y​′​​​γ​​log(1−y​′​​),y=0​​

α\alphaα仅仅是样本比例的调节因子。γ\gammaγ 是调节简单样本权重降低的因素，当 γ\gammaγ 为0时，就是交叉熵损失函数。

假设 γ=2\gamma=2γ=2 由上式可知，当y=1时，对于简单样本来说概率是很高的，假设是0.95。那么(1-0.95)^2 是非常小的。但是，对于难分类的样本来说，可能预测的是0.1，那么(1-0.1)^2 就很高了。所以会对难分类的样本给予更高的权重。

# Hinge loss

Hinge loss最适合分类问题，因为目标值在 {-1,1} 的集合中。它允许在实际值和预测值之间有符号差异时分配更多的错误。因此产生了比交叉熵更好的性能。

# Kullback Leibler Divergence Loss

KL散度损失计算概率分布和基础分布之间的divergence ，并找出以比特为单位丢失了多少信息。输出是非负值，指定两个概率分布有多接近。为了从概率的角度描述KL发散，使用了似然比。

# MSE

计算平方根误差损失

MSE=1nsmaple∑(yi−y^i)2\text{MSE}=\frac{1}{n_{\text{smaple}}}\sum(y_i-\hat{y}_i)^2 MSE=​n​smaple​​​​1​​∑(y​i​​−​y​^​​​i​​)​2​​

通常来说，利用均方差更容易求解，但平方绝对误差则对于异常点更鲁棒。当预测值与真实值很靠近时(小于1)，绝对值误差是小于平方值误差的；但是如果稍有点远(大于1)，那么平方值误差会更大。所以平方值误差能够更快的收敛。但是如果遇到异常值，平方值误差会努力让模型去拟合它，导致性能下降。
