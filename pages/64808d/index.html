Overloading the arithmetic operators using friend functions

# Overloading the arithmetic operators using friend functions

c++中最常用的一些运算符是算术运算符——即加号运算符(+)、减号运算符(-)、乘法运算符(*)和除号运算符(/)。注意，所有算术操作符都是二元操作符——这意味着它们接受两个操作数——操作符两边各一个。这四个操作符都以完全相同的方式重载。

事实证明，重载操作符有三种不同的方式:成员函数方式、友元函数方式和普通函数方式。在这一课中，我们将介绍友元函数的方法(因为它对大多数二进制运算符更直观)。下节课，我们将讨论普通函数法。最后，在本章后面的一节课中，我们将介绍成员函数的方式。当然，我们也会更详细地总结何时使用它们。

[[include]] <iostream>
 
class Cents
{
private:
	int m_cents;
 
public:
	Cents(int cents) { m_cents = cents; }
 
	// add Cents + Cents using a friend function
	friend Cents operator+(const Cents &c1, const Cents &c2);
 
	int getCents() const { return m_cents; }
};
 
// note: this function is not a member function!
Cents operator+(const Cents &c1, const Cents &c2)
{
	// use the Cents constructor and operator+(int, int)
	// we can access m_cents directly because this is a friend function
	return Cents(c1.m_cents + c2.m_cents);
}
 
int main()
{
	Cents cents1{ 6 };
	Cents cents2{ 8 };
	Cents centsSum{ cents1 + cents2 };
	std::cout << "I have " << centsSum.getCents() << " cents.\n";
 
	return 0;

重载plus运算符（+）就像声明一个名为operator +的函数一样简单，为它提供两个我们要添加的操作数类型的参数，选择一个合适的返回类型，然后编写该函数。

在Cents对象中，实现operator+()函数非常简单。首先是形参类型:在这个版本的operator+中，我们将把两个cent对象加在一起，因此函数将接受两个cent类型的对象。第二，返回类型:我们的操作符+将返回类型为Cents的结果，这就是我们的返回类型。

最后，实现: 要添加两个Cents对象，我们需要添加每个Cents对象的m_cents成员。因为重载的operator+()函数是该类的友元，所以可以直接访问形参的m_cents成员。另外，因为m_cents是一个整数，而且c++知道如何使用内置的+操作符(用于处理整数操作数)将整数相加，所以我们可以简单地使用+操作符来做加法。

# Friend functions can be defined inside the class

即使友元函数不是类的成员，如果需要，它们仍然可以在类内部定义:

[[include]] <iostream>
 
class Cents
{
private:
	int m_cents;
 
public:
	Cents(int cents) { m_cents = cents; }
 
	// add Cents + Cents using a friend function
        // This function is not considered a member of the class, even though the definition is inside the class
	friend Cents operator+(const Cents &c1, const Cents &c2)
	{
		// use the Cents constructor and operator+(int, int)
		// we can access m_cents directly because this is a friend function
		return Cents(c1.m_cents + c2.m_cents);
	}
 
	int getCents() const { return m_cents; }
};
 
int main()
{
	Cents cents1{ 6 };
	Cents cents2{ 8 };
	Cents centsSum{ cents1 + cents2 };
	std::cout << "I have " << centsSum.getCents() << " cents.\n";
 
	return 0;
}

# Overloading operators for operands of different types

通常情况下，您希望重载操作符与不同类型的操作数一起工作。例如，如果我们有Cents(4)，我们可能想要将整数6加到其中以产生结果Cents(10)。

当c++计算表达式x + y时，x成为第一个形参，y成为第二个形参。当x和y具有相同的类型时，添加x + y或y + x都没有关系——无论是哪种方式，都会调用相同版本的operator+。然而，当操作数具有不同类型时，x + y不会调用与y + x相同的函数。

例如，Cents(4) + 6调用operator+(Cents, int)，而6 + Cents(4)调用operator+(int, Cents)。因此，当重载不同类型的操作数的二进制操作符时，实际上需要编写两个函数——每种情况一个函数。下面是一个例子:

[[include]] <iostream>
 
class Cents
{
private:
	int m_cents;
 
public:
	Cents(int cents) { m_cents = cents; }
 
	// add Cents + int using a friend function
	friend Cents operator+(const Cents &c1, int value);
 
	// add int + Cents using a friend function
	friend Cents operator+(int value, const Cents &c1);
 
 
	int getCents() const { return m_cents; }
};
 
// note: this function is not a member function!
Cents operator+(const Cents &c1, int value)
{
	// use the Cents constructor and operator+(int, int)
	// we can access m_cents directly because this is a friend function
	return { c1.m_cents + value };
}
 
// note: this function is not a member function!
Cents operator+(int value, const Cents &c1)
{
	// use the Cents constructor and operator+(int, int)
	// we can access m_cents directly because this is a friend function
	return { c1.m_cents + value };
}
 
int main()
{
	Cents c1{ Cents{ 4 } + 6 };
	Cents c2{ 6 + Cents{ 4 } };
 
	std::cout << "I have " << c1.getCents() << " cents.\n";
	std::cout << "I have " << c2.getCents() << " cents.\n";
 
	return 0;
}

# Another example

[[include]] <iostream>
 
class MinMax
{
private:
	int m_min; // The min value seen so far
	int m_max; // The max value seen so far
 
public:
	MinMax(int min, int max)
	{
		m_min = min;
		m_max = max;
	}
 
	int getMin() const { return m_min; }
	int getMax() const { return m_max; }
 
	friend MinMax operator+(const MinMax &m1, const MinMax &m2);
	friend MinMax operator+(const MinMax &m, int value);
	friend MinMax operator+(int value, const MinMax &m);
};
 
MinMax operator+(const MinMax &m1, const MinMax &m2)
{
	// Get the minimum value seen in m1 and m2
	int min{ m1.m_min < m2.m_min ? m1.m_min : m2.m_min };
 
	// Get the maximum value seen in m1 and m2
	int max{ m1.m_max > m2.m_max ? m1.m_max : m2.m_max };
 
	return { min, max };
}
 
MinMax operator+(const MinMax &m, int value)
{
	// Get the minimum value seen in m and value
	int min{ m.m_min < value ? m.m_min : value };
 
	// Get the maximum value seen in m and value
	int max{ m.m_max > value ? m.m_max : value };
 
	return { min, max };
}
 
MinMax operator+(int value, const MinMax &m)
{
	// call operator+(MinMax, int)
	return { m + value };
}
 
int main()
{
	MinMax m1{ 10, 15 };
	MinMax m2{ 8, 11 };
	MinMax m3{ 3, 12 };
 
	MinMax mFinal{ m1 + m2 + 5 + 8 + m3 + 16 };
 
	std::cout << "Result: (" << mFinal.getMin() << ", " <<
		mFinal.getMax() << ")\n";
 
	return 0;
}

MinMax类跟踪到目前为止它看到的最小值和最大值。我们重载了+操作符3次，这样就可以将两个MinMax对象相加，或者向MinMax对象添加整数。

你将注意到的是我们添加到mFinal中的最小值和最大值。

让我们讨论一下MinMax mFinal = m1 + m2 + 5 + 8 + m3 + 16是如何计算的。记住，operator+的优先级高于operator=，并且operator+从左到右计算，所以m1 + m2先计算。这变成了对operator+(m1, m2)的调用，该调用产生返回值MinMax(8,15)。然后是MinMax(8,15) + 5。这变成了对operator+(MinMax(8,15)， 5)的调用，它产生返回值MinMax(5,15)。然后MinMax(5,15) + 8以同样的方式计算得到MinMax(5,15)。然后MinMax(5,15) + m3得到MinMax(3,15)。最后，MinMax(3,15) + 16等于MinMax(3,16)最后的结果被分配给mFinal。

换句话说，该表达式的计算结果为“MinMax mFinal = (((((m1 + m2) + 5) + 8) + m3) + 16)”，每个后续操作都返回一个MinMax对象，该对象成为下一个操作符的左操作数。

# Implementing operators using other operators

通常可以通过调用其他重载操作符来定义重载操作符。当这样做会产生更简单的代码时，您应该这样做。在实现很简单的情况下(例如，一行代码)，通常不值得这样做，因为添加一个额外的函数调用的间接方式比直接实现函数要复杂得多。
