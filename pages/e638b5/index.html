# Sigmoid函数

# 一元场景

logitcs 回归使用的拟合函数，其实是sigmoid函数

f(z)=11+e(−z)f(z)=\frac{1}{1+e^{(-z)}} f(z)=​1+e​(−z)​​​​1​​

sigmoid将一个(-\infin,+\infin)的值映射到(0,1)(0,1)(0,1)之间，更像是发生一件事的概率。

# 多元场景

这里，z不单单是一个变量，而是一个多元线性函数

z=β0+β1x1+⋯+βpxpz=\beta_0+\beta_1x_1+\cdots+\beta_px_p z=β​0​​+β​1​​x​1​​+⋯+β​p​​x​p​​

此时的sigmoid函数为：

f(z)=11+e−Xβf(z)=\frac{1}{1+e^{-X\beta}} f(z)=​1+e​−Xβ​​​​1​​

# 似然函数

一件事发生的可能性由概率来描述，但是，似然函数是衡量事情已经发生了，最有可能导致这些事情发生的情况(参数)是什么？

  • 已知参数的情况下，某件事发生的概率是P(x∣β)P(x|\beta)P(x∣β)。
  • 某件事情发生的情况下，参数的似然函数是L(β∣x)\mathbb{L}(\beta|x)L(β∣x)。

# 最大似然估计

在二分类问题中，y只取0和1，那么y出现的概率可以写成

P(y)=P(y=1)yP(y=0)1−yP(y)=P(y=1)^yP(y=0)^{1-y} P(y)=P(y=1)​y​​P(y=0)​1−y​​

加上参数的话，就是

P(y∣x,β)=P(y=1∣x,β)y[1−P(y=1∣x,β)]1−yP(y|x,\beta)=P(y=1|x,\beta)^y\left[1-P(y=1|x,\beta)\right]^{1-y} P(y∣x,β)=P(y=1∣x,β)​y​​[1−P(y=1∣x,β)]​1−y​​

接下来，我们将其带入到sigmoid函数中。我们知道在sigmoid函数中

P(y=1)=11+e−(xβ)P(y=1)=\frac{1}{1+e^{-(x\beta)}} P(y=1)=​1+e​−(xβ)​​​​1​​

P(y∣x,β)=(11+e−xβ)y(1−11+e−xβ)1−yP(y|x,\beta)=\left(\frac{1}{1+e^{-x\beta}}\right)^y\left(1-\frac{1}{1+e^{-x\beta}}\right)^{1-y} P(y∣x,β)=(​1+e​−xβ​​​​1​​)​y​​(1−​1+e​−xβ​​​​1​​)​1−y​​

对于多组数据{x1,yi},⋯,{xn,yn}\{x_1,y_i\},\cdots,\{x_n,y_n\}{x​1​​,y​i​​},⋯,{x​n​​,y​n​​}，希望在某个β∗\beta^*β​∗​​时，这些事情发生的概率的最大：

L(β)=∏i=1n(11+e−xiβ)yi(1−11+e−xiβ)1−yiL(\beta)=\prod_{i=1}^{n}\left(\frac{1}{1+e^{-x_i\beta}}\right)^{y_i}\left(1-\frac{1}{1+e^{-x_i\beta}}\right)^{1-y_i} L(β)=​i=1​∏​n​​(​1+e​−x​i​​β​​​​1​​)​y​i​​​​(1−​1+e​−x​i​​β​​​​1​​)​1−y​i​​​​

note：直接相乘建立在发生的事件是相互独立的。

当L(β)\mathbb{L}(\beta)L(β)取得最大时，此时的参数β\betaβ就是最优的。

直接求乘积会在计算上会比较麻烦，先将似然函数取对数，然后求导等于0。获得一个β\betaβ

\begin{equation} \begin{aligned} \log{(L(\beta))} &=\sum_{i=0}^{n}\left[y_i\log{\left(\frac{1}{1+e^{-x_i\beta}}\right)}+(1-y_i)\log{\left(1-\frac{1}{1+e^{-x_i\beta}}\right)}\right]\\ &= \sum_{i=0}^{n}\left[y_i\log{\sigma(x_i\beta)}+(1-y_i)\log{(1-\sigma(x_i\beta))}\right]\\ \frac{\partial{L(\beta)}}{\partial{\beta}}&=0\\ \end{aligned} \end{equation}

这里σ(x)\sigma(x)σ(x)是sigmoid函数。

如果用解析法求参数β\betaβ的话，需要一次性计算所有数据，需要耗费大量内存。所以还是选择梯度下降。

# 梯度下降求解损失

首先，我们知道

σ′(x)=σ(x)(1−σ(x))\sigma'(x)=\sigma(x)(1-\sigma(x)) σ​′​​(x)=σ(x)(1−σ(x))

并且

log′(σ(x))=1σ(x)∗(σ(x)∗(1−σ(x)))=(1−σ(x))log′(1−σ(x))=11−σ(x)∗(σ(x)∗(σ(x)−1))=−σ(x)\begin{aligned} \log'(\sigma(x))&=\frac{1}{\sigma(x)}*(\sigma(x)*(1-\sigma(x)))=(1-\sigma(x))\\ \log'(1-\sigma(x))&=\frac{1}{1-\sigma(x)}*(\sigma(x)*(\sigma(x) - 1))=-\sigma(x) \end{aligned} ​log​′​​(σ(x))​log​′​​(1−σ(x))​​​=​σ(x)​​1​​∗(σ(x)∗(1−σ(x)))=(1−σ(x))​=​1−σ(x)​​1​​∗(σ(x)∗(σ(x)−1))=−σ(x)​​

L(β)=∑i=0n[yilogσ(xiβ)+(1−yi)log(1−σ(xiβ))]L(\beta)=\sum_{i=0}^{n}\left[y_i\log{\sigma(x_i\beta)}+(1-y_i)\log{(1-\sigma(x_i\beta))}\right] L(β)=​i=0​∑​n​​[y​i​​logσ(x​i​​β)+(1−y​i​​)log(1−σ(x​i​​β))]

具体，损失对某个特征βi\beta_iβ​i​​的偏导为

∂L(β)∂βj=∑i=1n[yi∗(1−σ(xiβ)∗xij)−(1−yi)∗σ(xiβ)∗xij]=∑i=1n[yi∗xij−yi∗σ(xiβ)∗xij−σ(xiβ)∗xij+yi∗σ(xiβ)∗xij]=∑i=1n[yi∗xij−σ(xiβ)∗xij]=∑i=1n[yi−σ(xiβ)]∗xij\begin{aligned} \frac{\partial{L(\beta)}}{\partial{\beta_j}}&= \sum_{i=1}^{n}\left[y_i*(1-\sigma(x_i\beta)*x_{ij}) - \right(1-y_i)*\sigma(x_i\beta)*x_{ij}]\\ &=\sum_{i=1}^{n}[y_i*x_{ij}-y_i*\sigma{(x_i\beta)}*x_{ij}-\sigma(x_i\beta)*x_{ij}+y_i*\sigma(x_i\beta)*x_{ij}]\\ &=\sum_{i=1}^{n}\left[y_i*x_{ij}-\sigma(x_i\beta)*x_{ij}\right]\\ &=\sum_{i=1}^{n}[y_i-\sigma(x_i\beta)]*x_{ij} \end{aligned} ​​∂β​j​​​​∂L(β)​​​​​​​​=​i=1​∑​n​​[y​i​​∗(1−σ(x​i​​β)∗x​ij​​)−(1−y​i​​)∗σ(x​i​​β)∗x​ij​​]​=​i=1​∑​n​​[y​i​​∗x​ij​​−y​i​​∗σ(x​i​​β)∗x​ij​​−σ(x​i​​β)∗x​ij​​+y​i​​∗σ(x​i​​β)∗x​ij​​]​=​i=1​∑​n​​[y​i​​∗x​ij​​−σ(x​i​​β)∗x​ij​​]​=​i=1​∑​n​​[y​i​​−σ(x​i​​β)]∗x​ij​​​​

现在，把损失对某个参数βj\beta_jβ​j​​的梯度求出来了，那么梯度下降，需要在添加一个负号。然后更新参数

也就是

−∂L∂βj=∑i=1n[σ(xiβ)−yi]∗xij-\frac{\partial{L}}{\partial{\beta_j}}=\sum_{i=1}^{n}[\sigma(x_i\beta)-y_i]*x_{ij} −​∂β​j​​​​∂L​​=​i=1​∑​n​​[σ(x​i​​β)−y​i​​]∗x​ij​​

  1. 初始化参数向量β\betaβ的值。
  2. 以步长α\alphaα乘损失对参数的负梯度，获得更新的步长
  3. β1=β0−α∗∇\beta_1=\beta_0-\alpha*\nablaβ​1​​=β​0​​−α∗∇
  4. 重复训练，以至于收敛。
